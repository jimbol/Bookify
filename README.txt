The files herein are reliant on jQuery 1.3 and higher.

_bookify1.4.2.js holds Booky object which creates the core functionality of the eBook.
_script.js houses the other core js including hover events, preloader handler, window resize event and a check for pannability of the eBook.
_zoom.js holds the zoom functionality, switching between stylesheets, and sensing the window size.
popup.js holds the modal code.
